6. 各檔案說明：
	b11107035-p3/main.cpp		: main source code
	b11107035-p3/io.cpp		: cpp file for all input and output functions
	b11107035-p3/picRouting.cpp	: cpp file for all other routing functions
	b11107035-p3/picRouting.h	: header file contains data structures
	b11107035-p3/Makefile		: makefile
	b11107035-p3/readme.txt		: this file
	b11107035-p3/report.		: report file explaining data structures and algorithms 
7. 編譯方式說明： 
	open the the folder b11107035-p3/ in terminal and run the command 'make'.
	it will automatically compile the code and generate the 'picRouting' executable file.
8. 執行、使用方式說明：
	after compiling, make sure the 'picRouting' executable file has been generated.
	also, make sure there is an input file in the same folder.
	then, use the command './picRouting [input file name] [output file name]' to run.
	(ex: ./picRouting pic5x5.in output.txt)
	after running the code, it should output information about losses in terminal, and generate an output file (.txt) in the same folder.
