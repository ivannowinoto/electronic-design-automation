# Electronic Design Automation (EDA) - Course Projects

A comprehensive collection of programming assignments and competition solutions from an undergraduate Electronic Design Automation course, showcasing various EDA algorithms and methodologies implemented in C++.

## Overview

This repository contains four major projects covering fundamental topics in Electronic Design Automation: visualization, placement, routing, and RTL design automation. Each project demonstrates practical implementations of industry-standard algorithms used in modern chip design workflows.

## Repository Structure

```
electronic-design-automation/
├── 01-file-io-visualization/         # Visualization tool implementation
├── 02-standard-cell-legalizer/       # Cell legalization
├── 03-photonic-detailed-router/      # Photonic IC detailed routing
├── 04-cad-contest-problem-E/         # RTL reconnection (ICCAD 2025)
└── README.md                         # This file
```

Each folder contains:
- Complete source code implementation
- Makefile for compilation
- Detailed README.md with usage instructions
- Problem summary and specifications
- Test cases (where applicable)

## Technical Specifications

- **Language**: C/C++
- **Platform**: Linux/Unix
- **Build System**: GNU Make
- **Compiler**: g++ (C++ standard)

## Quick Start

1. Clone the repository:
```bash
git clone <repository-url>
cd electronic-design-automation
```

2. Navigate to any project:
```bash
cd <folder-name>
```

3. Build and run:
```bash
make
./<executable> <input-files> <output-file>
```

Refer to each folder's README.md for specific instructions.

## Academic Context

**Projects 1-3**: Programming assignments from undergraduate EDA course  
**Project 4**: ICCAD 2025 CAD Contest - Problem E submission

- Graph algorithms and pathfinding
- Physical design automation
