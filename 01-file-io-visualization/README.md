# Programming Assignment 1: Physical Circuit Design Visualization

## Overview

This project implements a visualization tool for physical circuit designs. The program processes circuit layout data and generates Gnuplot scripts to visualize the physical placement of circuit components.

## Requirements

- **Language:** C++
- **Platform:** Linux
- **Compiler:** GNU g++
- **Visualization:** Gnuplot (for rendering output)

## Project Structure

```
.
├── main.cpp          # Main program entry point
├── genPlot.cpp       # Core visualization functions
├── genPlot.h         # Data structures and function declarations
├── Makefile          # Build automation
├── README.md         # Project documentation
├── pa1.txt           # Problem statement summary
├── pa1_testcases/    # Directory containing test input files
└── output_sample/    # Example PNG outputs generated via Gnuplot
```

## Building the Project

### Compilation

Navigate to the project directory and run:

```bash
make
```

This will compile all source files and generate the executable `genPlot`.

### Cleaning Build Files

To remove compiled objects and executables:

```bash
make clean
```

## Usage

### Command Syntax

```bash
./genPlot [msbcs_width] [msbcs_height] [input_file] [output_file]
```

### Parameters

| Parameter | Description |
|-----------|-------------|
| `msbcs_width` | Width of the MSBCS (Minimum Separation Between Cell Sites) |
| `msbcs_height` | Height of the MSBCS |
| `input_file` | Input file containing circuit layout data (e.g., `input.txt`) |
| `output_file` | Output Gnuplot script file (e.g., `output.gp`) |

### Example Workflow

#### Step 1: Generate Gnuplot Script

```bash
./genPlot 7100 6600 input.txt output.gp
```

This command:
- Sets MSBCS width to 7100 units
- Sets MSBCS height to 6600 units
- Reads circuit data from `input.txt`
- Generates Gnuplot script `output.gp`

#### Step 2: Render Visualization

```bash
gnuplot output.gp
```

This produces a PNG image visualizing the circuit layout based on the generated script.

## Input/Output Format

### Input
The program accepts text files containing circuit layout specifications, including component positions and dimensions.

### Output
- **Gnuplot Script (.gp):** A script file that defines the visualization parameters
- **PNG Image:** Generated by Gnuplot from the script file (see `output_sample/` for examples)

## Troubleshooting

### Gnuplot not found
Ensure Gnuplot is installed and accessible in your system PATH.

### Output image not generated
Check that the `.gp` file was created successfully and contains valid Gnuplot commands.
